\# Setup Guide



\## Prerequisites

\- Airtable base with Candidates table

\- Make.com account

\- PDF.co API key

\- Groq API key



\## Schema Requirements

\- Full Name (Single line text)

\- CV Attachment (Attachment)

\- Automation Status (Single select: Ready, Processing, Done)

\- AI Summary (Long text)

\- AI Match Score (Number)

\- Last Modified Time (Last modified time)



\## Import

1\. Export blueprint from Make.com

2\. Import into new scenario

3\. Reconnect all modules

4\. Set trigger to watch Last Modified Time field
